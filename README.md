# RAT
## Note 
* Client is server, server is client
* Use C++17
## How to test
* Client: open server directory in terminal
* Server: open client directory in terminal
```
make debug
```
## How to use
* Kill process (like Terminal)
```
kill <pid>
```
or
```
killall <process name>
```
* List process (like Terminal)
```
ps -e
```
* Get a file
```
getfile <file path>
```
